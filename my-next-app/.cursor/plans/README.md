# Plans

이 폴더는 Cursor Agent가 생성한 구현 계획을 저장하는 공간입니다.

## 사용법

1. 복잡한 작업을 시작하기 전에 `Shift+Tab`으로 Plan Mode를 켜세요
2. 에이전트가 계획을 생성하면 **"Save to workspace"** 를 클릭하면 이 폴더에 저장됩니다
3. 저장된 계획은 나중에 `@.cursor/plans/[파일명]` 으로 참조할 수 있습니다

## 네이밍 컨벤션

```
YYYY-MM-DD-feature-name.md
예: 2026-04-06-add-user-auth.md
```

## 팁

- 중단된 작업을 재개할 때 이전 플랜을 컨텍스트로 제공하세요
- 같은 기능을 작업하는 팀원과 플랜을 공유하세요
- 플랜이 틀렸으면 직접 수정 후 에이전트를 다시 실행하세요
